CloudScale Cleanup — Free WordPress Database Cleaner, Image Optimizer & Cron Manager

WP-Optimize and Advanced Database Cleaner charge $39–$99/year for database cleanup that should be free. CloudScale Cleanup gives you a full WordPress database optimiser: delete post revisions, expired transients, auto-drafts, trashed posts, spam comments, and orphaned post metadata — with a dry-run preview showing exactly what will be deleted before you commit. Includes a media library orphan scanner, bulk image optimisation (lossy and lossless), PNG-to-JPEG conversion to reduce file sizes by up to 70%, and a live WP-Cron management console to inspect, run, reschedule, or delete any scheduled event. All operations logged. Completely free, open source, no subscription, no premium tier.
